Privacy Policy — CatchTheBall

CatchTheBall is a simple mobile game created for entertainment purposes. This app does not collect, store, or share any personal information from users.

1. Data Collection

The game does not collect:

  • Names
  • Emails
  • Phone numbers
  • Device identifiers
  • Location
  • Any personal or sensitive data

2. Local Storage

The game only saves:

  • Best Score (locally on the device)

This data stays on the user’s device and is not shared with anyone.

3. Children Privacy

The game is suitable for all ages. No personal data is collected from children or adults.

4. Third‑Party Services
